````markdown
# FoodLink Web Frontend

Giao diện người dùng cho nền tảng web của dự án FoodLink, được xây dựng bằng Vite.

## 🚀 Bắt đầu

Làm theo các bước dưới đây để cài đặt và chạy dự án trên máy của bạn.

### 📋 Yêu cầu cài đặt

- Node.js (phiên bản LTS)
- Yarn hoặc npm

### 🔧 Cài đặt

1.  **Clone repository**

    ```sh
    git clone <your-web-frontend-repo-url>
    cd web-frontend-foodlink
    ```

2.  **Cài đặt dependencies**
    Sử dụng npm:
    ```sh
    npm install
    ```
    Hoặc sử dụng Yarn:
    ```sh
    yarn install
    ```

### 🏃 Chạy dự án

Để khởi động server phát triển với Vite:

```sh
npm run dev
```
````
